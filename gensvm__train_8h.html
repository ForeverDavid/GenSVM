Utility function for training a GenSVM model.

This function organizes model allocation, kernel preprocessing, instance weight initialization, and model training. It is the function that should be used for training a single GenSVM model. Note that optionally a seed model can be passed to the function to seed the V matrix with. When no such model is used this parameter should be set to NULL.

Parameters
[in]modela GenModel instance
[in]dataa GenData instance with the training data
[in]seed_modelan optional GenModel to seed the V matrix
